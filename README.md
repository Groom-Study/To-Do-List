## 📋 팀 미션: TO-DO-LIST
웹 애플리케이션의 전체 구조를 이해하고, 프론트엔드에서 만든 To Do 앱 UI를 Express와 MongoDB 같은 적용이 쉬운 백엔드 스택을 참고하여, 간단한 서버와 데이터 저장 구조로 확장해봅니다. 백엔드 구현은 실제 기술을 깊이 다루기보다는, 기초적인 연동 흐름과 REST API의 개념 적용에 초점을 둡니다.

(습득 역량) ● 간단한 Express 서버 구조 이해 및 적용 ● REST API와 프론트-백 연동 흐름 경험 ● React 상태 관리 및 컴포넌트 설계 ● 클라이언트-서버 간 데이터 흐름 구조화 ● 기초적인 전체 웹 애플리케이션 구성 경험

(결과물) ● 전체 페이지 화면설계서 ● 연동 흐름을 포함한 시스템 구조 문서 ● To Do 앱 결과물 레포 URL (백엔드 연동 포함)
